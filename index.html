To get all the FastAPI and SQLModel code for the course described in the video, you can click here to download a zip file. Or grab it from github at https://github.com/AlanSimpsonMe/FastAPI-SQLModel

To use the code, you'll need to be using Python 3.6 or later (the course is all 3.10). You'll also need to create your own virtual envronment. The first few lessons explain how. Or, if you're familiar with Python and virtual environments, you can set up the virtual environment with the usual command line stuff below. Keeping in mind that you should already know how to use Python 3.6 on your system.
